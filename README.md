# code-master# Code Master (C++ Programming Quiz System)

## 📌 Overview
Code Master is a console-based quiz application built in C++.  
It is designed to test and improve programming knowledge through different types of questions.

---

## ⚙️ Features
- Syntax error identification  
- Output prediction questions  
- Fill-in-the-blank programming tasks  
- Score calculation and result display  
- File handling for storing questions and results  

---

## 🛠️ Technologies Used
- C++  
- Object-Oriented Programming (OOP)  
- File Handling
